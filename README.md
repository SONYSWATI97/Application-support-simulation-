# Application Support & Incident Management Simulation

## 📌 Overview
This project is a Python-based simulation of a real-world application support system. It allows users to create, track, and resolve incidents, mimicking how production support teams manage application issues.

The system demonstrates key concepts such as incident tracking, issue prioritization, and resolution workflows used in IT support and NOC environments.

---

## 🚀 Features
- Create and manage incidents
- Track issue status (OPEN / RESOLVED)
- Assign priorities (High / Medium / Low)
- Store issues using JSON (simulates a database)
- Simple command-line interface

## 🛠️ Technologies Used
